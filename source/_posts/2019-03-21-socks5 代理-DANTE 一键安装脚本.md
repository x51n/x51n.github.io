## 一键安装

* 下载脚本

```
wget --no-check-certificate https://raw.github.com/Lozy/danted/master/install.sh -O install.sh
```

* 安装

```
bash install.sh  --port=端口 --user=自己设置用户名 --passwd=设置密码
```

* 结束

成功后会出现：Dante Server Install Successfuly!

显示用户名和密码
